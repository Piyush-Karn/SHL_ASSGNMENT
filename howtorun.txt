=======================================================
HOW TO RUN THE EVALUATION AND TESTS LOCALLY
=======================================================

Since the evaluation script requires the API server to be running in the background, you'll need to use two separate terminal windows.

---
TERMINAL 1: START THE API SERVER
---
1. Open your first terminal and ensure you are in the project root directory (`D:\SHL ASSIGNMENT`).
2. (Optional) Activate your Python virtual environment if you are using one.
3. Make sure your dependencies are installed:
   pip install -r requirements.txt

4. Start the FastAPI server locally:
   uvicorn app.main:app --host 127.0.0.1 --port 8000

   > Leave this terminal open and running! It might take a few seconds to load the dataset and build the indices. Wait until you see "Application startup complete."


---
TERMINAL 2: RUN THE TESTS & EVALUATION
---
1. Open a second terminal (while Terminal 1 is still running the server).
2. Ensure you are in the project root directory (`D:\SHL ASSIGNMENT`).

3. Run the fast smoke tests to ensure basic API health and endpoints are working:
   python tests/smoke_test.py

4. Run the full evaluation harness to test the recall against the 10 sample conversations:
   python -m evaluation.eval_harness --api-url http://127.0.0.1:8000

   > This will print out the detailed summary of the recall scores, missing assessments, and behavior probes!

=======================================================
When you are done testing, you can go back to Terminal 1 and press CTRL+C to stop the API server.
